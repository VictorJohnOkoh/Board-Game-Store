Gives a Command Line Interface for a Board Game Store that allows the user to log in as an admin or customer <br/>

Admins can:
* Add a board or video game
* View all items in stock (including their purchase price)
  
Customers can:
* View all items in stock (excluding their purchase price)
* Filter items based on compatibility or product ID
* Add a product to their basket
* Checkout with their basket
